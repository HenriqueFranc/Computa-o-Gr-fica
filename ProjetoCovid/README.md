# Covid Destroyer

Jogo arcade de uma vacina contra o coronavírus feito em OpenGL.

| Tecla  | Função                     |
| :----- | :------------------------- |
| 1      | Reinicia o Jogo            |
| ESC    | Sair do Jogo e fecha a tel |
| Setas  | Mover Vacina               |
| Espaço | Atirar                     |
